嘛是 Dabr?

  • 使用 Twitter API 的基于 Web 的客户端。
  • 一个开源的项目，原始版本是 @davidcarrington 带着 @whatleydude 的灵感，在牛x的 Terence Eden 的帮助下完成的。
  • 比较安全，因为所有登录信息都作为一个加密的 cookie 存在你的机器上。（服务器从来就不屑于存储这些玩意儿嘛！）

如果有啥建议或者意见就直说吧 =。=
